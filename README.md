\# Terraform First Project



This is my first Terraform project.



It uses the Docker provider to:

\- pull the nginx image

\- create a Docker container

\- expose the container on localhost:8080



\## Tools Used

\- Terraform

\- Docker Desktop



\## Files

\- `main.tf` - Terraform configuration

\- `.gitignore` - excludes Terraform state and provider files



\## How to Run



```bash

terraform init

terraform plan

terraform apply
